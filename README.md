# Record Shop API

A RESTful backend API for managing a music record shop, built with ASP.NET Core and Entity Framework.

## Overview
This API allows users to browse, search, add, update, and delete music records through secure, authenticated endpoints.

## Tech Stack
- C# / ASP.NET Core
- Entity Framework Core
- SQL Server
- NUnit / Moq (TDD)

## Features
- Full CRUD operations for records and artists
- JWT authentication
- Repository pattern and service layer
- Async/await throughout
- Unit tested with NUnit and Moq
